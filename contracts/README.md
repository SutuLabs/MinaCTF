## How to build

```sh
npm run build:docker
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```

## Dependencies

- PocketBase 0.16.7

## Important Notes

Don't forget to recompile the contract by `npm run compile` when 1) changed contract code and 2) upgraded framework version.
